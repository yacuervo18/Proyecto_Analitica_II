## Dashboard EDA — Seguro de Salud

Este proyecto contiene un tablero interactivo desarrollado en Streamlit para visualizar y presentar el Análisis Exploratorio de Datos (EDA) del seguro de salud.

### ¿Qué encontrarás en este dashboard?
- Visualizaciones clave del perfil demográfico, uso del seguro, costos, condiciones preexistentes, tipos de reclamación, análisis geográfico y correlaciones.
- Un entorno amigable para explorar los principales hallazgos del EDA.

---

### ¿Cómo ejecutar el dashboard?
1. Abre una terminal en la raíz del proyecto.
2. Ejecuta el siguiente comando:

```bash
streamlit run streamlit-eda-app/src/app.py
```

---

### Requisitos
- Python 3.8+
- Instalar dependencias con:

```bash
pip install -r streamlit-eda-app/requirements.txt
```

---

### Estructura relevante
- `src/app.py`: Script principal del dashboard.
- `src/data/`: Carpeta donde deben estar las imágenes del EDA.
